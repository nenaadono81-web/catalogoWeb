Bienvenido al Catálogo de Películas

Esta aplicación web fue desarrollada con Angular, Laravel y MySQL para administrar un catálogo de películas de manera dinámica y organizada.

El sistema permite visualizar películas, consultar información detallada, agregar nuevos registros, editar películas existentes y eliminar registros almacenados en la base de datos.

Además, la aplicación utiliza Bootstrap para proporcionar una interfaz moderna, responsiva y fácil de utilizar.
